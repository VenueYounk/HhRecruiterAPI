Recruiter API

GET /companies

Возращает компании с актуальными вакансиями и номерами телефонов, обновляется каждую ночь

page: (optional) Номер страницы результатов (по умолчанию: 1)
results_per_page: (optional) Количество результатов на странице (по умолчанию: 50)
sort_by: (optional) Поле для сортировки (по умолчанию: industries)
search_text: (optional) Текст для поиска по названию компании или отраслям

Response:

[
  {
    "company_name": "Компания А",
    "industry": "Технологии",
    "vacancies": [
      {
        "name": "Инженер-программист",
        "contact_name": "Иван Иванов",
        "phone": "123-456-7890"
      },
      {
        "name": "Аналитик данных",
        "contact_name": "Мария Сидорова",
        "phone": "987-654-3210"
      }
    ]
  },
  {
    "company_name": "Компания Б",
    "industry": "IT-услуги",
    "vacancies": [
      {
        "name": "Системный администратор",
        "contact_name": "Петр Петров",
        "phone": "555-123-4567"
      }
    ]
  }
]

GET /phones

Возращает компании с номерами телефонов, которые когда либо существовали в базе данных, ни один номер не уйдет

page: (optional) Номер страницы результатов (по умолчанию: 1)
results_per_page: (optional) Количество результатов на странице (по умолчанию: 50)
sort_by: (optional) Поле для сортировки (по умолчанию: name)
search_text: (optional) Текст для поиска по названию компании или отраслям
